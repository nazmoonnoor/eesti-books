## 6. Võrdlus: julgem kui karu
*(page 123 — the comparative "Missugune?": **II vorm + -m**)*

| I vorm | II vorm | Võrdlusaste *(II vorm + -m)* |
|---|---|---|
| julge | julge | julge**m** |
| kiire | kiire | kiire**m** |
| suur | suure | suure**m** |
| rumal | rumala | rumala**m** |
| tugev | tugeva | tugeva**m** |
| tark | targa | targe**m** |
| nõrk | nõrga | nõrge**m** |
| arg | ara | are**m** |
| laisk | laisa | laise**m** |
| väike | väikse | väikse**m** |
| aeglane | aeglase | aeglase**m** |
| töökas | tööka | tööka**m** |

> 🔹 *Jänes on **nõrgem kui** siil, aga siil on **targem kui** jänes.*
>
> Compare the pairs: tark–nõrk · rumal–aeglane · julge–töökas · arg–kiire · väike–tugev · suur–laisk.
