## 2. Jaatav → eitav
*(page 116 — the past negative: **ei + -nud**. Same form for every person.)*

The affirmative past uses **-sin / -in** (ma jõin, ma pesin). The **negative** past is **ei** + the **-nud** participle — and it never changes for person: *ma / sa / ta / me / te / nad **ei joonud***.

| Verb | Jaatav *(I did …)* | Eitav *(I didn't …)* |
|---|---|---|
| jooma | jõin | ei joonud |
| sööma | sõin | ei söönud |
| pesema | pesin | ei pesnud |
| magama | magasin | ei maganud |
| nägema | nägin | ei näinud |
| minema | läksin | ei läinud |
| tegema | tegin | ei teinud |
| lugema | lugesin | ei lugenud |
| käima | käisin | ei käinud |
| ärkama | ärkasin | ei ärganud |
| ostma | ostsin | ei ostnud |
| laulma | laulsin | ei laulnud |
| tantsima | tantsisin | ei tantsinud |
| istuma | istusin | ei istunud |
| helistama | helistasin | ei helistanud |
| vaatama | vaatasin | ei vaadanud |
| õppima | õppisin | ei õppinud |
| rääkima | rääkisin | ei rääkinud |
| kirjutama | kirjutasin | ei kirjutanud |
| töötama | töötasin | ei töötanud |
| koristama | koristasin | ei koristanud |
| maksma | maksin | ei maksnud |
| kuulama | kuulasin | ei kuulanud |
| sõitma | sõitsin | ei sõitnud |
| mängima | mängisin | ei mänginud |
| triikima | triikisin | ei triikinud |
| parandama | parandasin | ei parandanud |
| jalutama | jalutasin | ei jalutanud |

> 🔹 *Ma **jõin** eile õlut, aga ma **ei joonud** kohvi. Ma **koristasin** kööki, aga ma **ei koristanud** tuba.*

## 4. Jõuluvana lugu
*(page 117 — "correct the rumour": Ei käinud …! Ta käis …)*

> — Kas sa tead, et **jõuluvana käis eile Eestis!**
> — Mis asja? **Ei käinud Eestis!** Ta käis Soomes. Aga kas sa tead, et jõuluvana **käis restoranis?**
> — Mis asja? **Ei käinud restoranis!** Ta käis karaokebaaris. Aga kas sa tead, et jõuluvana …?

| ❌ Vale jutt *(rumour)* | ✅ Õige *(truth)* |
|---|---|
| käis Eestis · käis restoranis | käis Soomes · käis karaokebaaris |
| jõi 2 pudelit valget veini | jõi 2 pudelit vett |
| sõi 4 hamburgerit | sõi 5 hamburgerit |
| helistas Eesti presidendile | helistas taksojuhile |
| ostis sportauto | ostis lennuki |
| jalutas loomaaias | jalutas vanalinnas |
| rääkis minuga | rääkis sinu emaga |
