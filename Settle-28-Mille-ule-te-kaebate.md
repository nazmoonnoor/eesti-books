## 1. Küsi ja vasta
*(page 154 — ask about symptoms)*

> Kas su … valutab? · Kas sa oled …? · Kas sul on …?

## 2. Millal? (aeg)
*(page 154 — time expressions: "…tagasi" = ago, "…pärast" = in/after)*

| Minevik *(-tagasi = ago)* | Tulevik *(-pärast = in)* |
|---|---|
| kaks minutit tagasi | kahe minuti pärast |
| kaks tundi tagasi | kahe tunni pärast |
| kaks päeva tagasi | kahe päeva pärast |
| kaks nädalat tagasi | kahe nädala pärast |
| kaks kuud tagasi | kahe kuu pärast |
| kaks aastat tagasi | kahe aasta pärast |

> 🔹 Also: **väga ammu** *(long ago)* · **praegu** *(now)*.

## 3. Haigeks jääma → terveks saama
*(page 155 — "to fall ill" → "to get well"; -ks + past)*

| haigeks jääma *(fall ill)* | terveks saama *(get well)* | Lõpp |
|---|---|---|
| Ma jäin haigeks | Ma sain terveks | **-n** |
| Sa jäid haigeks | Sa said terveks | **-d** |
| Ta jäi haigeks | Ta sai terveks | **–** |
| Me jäime haigeks | Me saime terveks | **-me** |
| Te jäite haigeks | Te saite terveks | **-te** |
| Nad jäid haigeks | Nad said terveks | **-d** |

> 🔹 Eitus: *Ma **ei jäänud** haigeks. Ma **ei saanud** terveks.*

## 4. Arsti juures (dialoog)
*(pages 155–156 — "Mille üle te kaebate?"; doctor & patient)*

> **Arst:** Tere! Mille üle te kaebate?
> **Patsient:** Mul on köha ja nohu.
> **Arst:** Kas teil on palavik ka?
> **Patsient:** Jah. 38,2 kraadi, täpsemalt 38,2.
> **Arst:** Oi-oi! Ja millal te haigeks jäite?
> **Patsient:** Vist kaks päeva tagasi. Ei, tegelikult juba … . Vist.
> **Arst:** Selge! Tehke suu lahti! Kurk on punane. Kas pea valutab ka?
> **Patsient:** Valutab. Ja pea valutab.
> **Arst:** Vaatame. Siin on väike põletik. Teeme analüüsid. Ja ma kirjutan teile ravimi.
> **Patsient:** Aitäh!

## 5. Käsk ja keeld (-ge/-ke)
*(pages 156–157 — the polite / plural imperative: **-ge / -ke** ; keeld = **Ärge …!**)*

The polite (or plural "teie") command adds **-ge / -ke** to the verb stem; the prohibition is **Ärge …!**

| Kas ma võin …? | Käsk *(do!)* | Keeld *(don't!)* |
|---|---|---|
| … aias töötada? | Töötage! | Ärge töötage! |
| … päeval magada? | Magage! | Ärge magage! |
| … linnas jalutada? | Jalutage! | Ärge jalutage! |
| … rõdul päevitada? | Päevitage! | Ärge päevitage! |
| … ujuda? | Ujuge! | Ärge ujuge! |
| … raamatut lugeda? | Lugege! | Ärge lugege! |
| … rääkida? | Rääkige! | Ärge rääkige! |
| … reisida? | Reisige! | Ärge reisige! |
| … tantsida? | Tantsige! | Ärge tantsige! |
| … laulda? | Laulge! | Ärge laulge! |
| … korterit koristada? | Koristage! | Ärge koristage! |
| … šokolaadi süüa? | Sööge! | Ärge sööge! |
| … õlut juua? | Jooge! | Ärge jooge! |
| … saunas käia? | Käige! | Ärge käige! |
| … perega koos olla? | Olge! | Ärge olge! |
| … homme tööle minna? | Minge! | Ärge minge! |
| … televiisorit vaadata? | Vaadake! | Ärge vaadake! |
| … aknaid pesta? | Peske! | Ärge peske! |
| … jalgrattaga sõita? | Sõitke! | Ärge sõitke! |
| … sporti teha? | Tehke! | Ärge tehke! |

> 🔹 **Mida sinu vanaema ütleb?** *(page 157)* — *Mul on nohu.* → **Jooge teed! Magage! Ärge käige väljas!** *Ma olen kogu aeg väsinud.* → **Puhake! Ärge töötage nii palju! Magage rohkem!**
