## 3. Käin tööl / Lähen tööle
*(page 79 — **Kus?** (-l / -s, location) vs **Kuhu?** (-le / -sse, direction))*

| Kus? *(käin …)* | Kuhu? *(lähen …)* |
|---|---|
| käin töö**l** | lähen töö**le** |
| käin turu**l** | lähen turu**le** |
| käin koosoleku**l** | lähen koosoleku**le** |
| käin lõuna**l** | lähen lõuna**le** |
| käin teatri**s** | lähen teatri**sse** |
| käin kohviku**s** | lähen kohviku**sse** |
| käin poe**s** | lähen **poodi** |
| käin kino**s** | lähen **kinno** |
| käin restorani**s** | lähen **restorani** |
| käin trenni**s** | lähen **trenni** |
| käin sõbra**l** küla**s** | lähen sõbra**le** **külla** |
| käin ülikooli**s** | lähen **ülikooli** |

> 🔹 **Kus?** = where you *are* (käin = I go regularly / attend). **Kuhu?** = where you're *going to*. *Esmaspäeval käin eesti keele kursusel. Ma lähen eesti keele kursusele kell kaks.*

## 4. Esmaspäeviti
*(pages 80–82 — "on Mondays …": the **-ti** form)*

| Nädalapäev | … -ti *(regularly)* |
|---|---|
| esmaspäev | esmaspäeviti |
| teisipäev | teisipäeviti |
| kolmapäev | kolmapäeviti |
| neljapäev | neljapäeviti |
| reede | reedeti |
| laupäev | laupäeviti |
| pühapäev | pühapäeviti |

> 🔹 Frequency: **iga päev** *(every day)* · **… korda nädalas** *(… times a week)* · **esmaspäeviti** *(on Mondays)*. Connectors: **siis** *(then)* · **seejärel** *(after that)* · **pärast seda** *(after that)*.
>
> *Esmaspäeviti käin trennis. Romeo käib kaks korda nädalas trennis. Reede õhtul olen alati kodus.*
